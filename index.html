Common Cybersecurity Threats

Phishing

Phishing is a scam where attackers trick you into revealing personal info via fake emails or sites.

Learn More

Phishing often uses emails that look legitimate, asking for passwords or personal data. Always verify sender details and never click suspicious links.

Malware/Ransomware

Malware is malicious software that damages or steals data. Ransomware encrypts your files and demands payment for decryption.

Learn More

Keep software updated and use antivirus. Backup data regularly to recover from ransomware without paying the attacker.

Social Engineering

Social engineering tricks people into revealing secrets. It relies on human trust (like pretending to be a co-worker).

Learn More

Be cautious if someone pressures you to act quickly or bypass security. Always verify identities before sharing info.

Password Hygiene

Good password hygiene means using strong, unique passwords and changing them regularly to protect accounts.

Learn More

Use a mix of letters, numbers, and symbols. Avoid reusing passwords. A password manager can help keep track of them securely.

Data Backups

Data backups involve copying important information to a secure location so you can restore it if it’s lost or locked.

Learn More

Use external drives or cloud services to back up files. The 3-2-1 rule helps: 3 copies, 2 types of media, 1 off-site backup.
